# KjumpBLE
## 目的
Validic 與 CVS 合作將各個廠牌的藍芽產品通訊方式都加進 CVS 的 APP 裡面，讓 CVS 的 APP 可以順利與他們所購買的產品做溝通，而我們公司的產品也是 CVS 的產品之一，因此 CVS 提出要讓我們將藍芽產品嵌入 Validic 的 Libraries 內以至於 CVS 的 APP 也能操作我們的產品。
